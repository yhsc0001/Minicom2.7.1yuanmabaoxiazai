## 关于 Minicom

Minicom 是一个基于文本的串行通信程序，广泛用于 Linux 和其他类 Unix 系统中。它主要用于与串行设备（如调制解调器、嵌入式设备等）进行通信。Minicom 提供了丰富的功能，包括文件传输、脚本执行、以及对多种串行协议的支持。
